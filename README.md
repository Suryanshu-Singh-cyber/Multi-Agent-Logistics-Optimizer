# 🚀 Nexus-Route

Multi-Agent Logistics Optimizer using Reinforcement Learning.

## 🔥 Features
- Multi-agent system (5 delivery agents)
- Reinforcement Learning (PPO)
- Decentralized coordination
- Streamlit visualization

## 🧠 Tech Stack
- Python
- Gymnasium
- Stable-Baselines3
- Streamlit

## 🚀 Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
